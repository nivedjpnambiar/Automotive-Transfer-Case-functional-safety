# V-Model Mapping

| Development artifact | Verification artifact |
|---|---|
| Item Definition | Acceptance review |
| Hazard analysis | Safety review |
| Safety goals | Safety validation |
| Functional requirements | System tests |
| Technical requirements | Integration tests |
| Software requirements | Unit tests |
| Python implementation | Code review and execution |

Each technical requirement traces upward to a safety goal and downward to an automated test.
